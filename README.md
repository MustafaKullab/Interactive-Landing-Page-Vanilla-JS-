
A responsive and interactive landing page built using **HTML, CSS, and Vanilla JavaScript**.  
This project focuses on practicing core frontend development concepts such as DOM manipulation, events handling, and building interactive UI components.

---

## 🚀 Live Demo

👉 https://interactive-landing-page-js.vercel.app/

---

## ✨ Features

- Responsive landing page design
- Interactive navigation / UI behavior
- DOM manipulation using JavaScript
- Event-driven user interactions
- Clean and structured layout
- Smooth user experience

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6 - Vanilla JS)

---

## 🎯 Purpose of the Project

This project was created as a hands-on practice to improve:

- Structuring real-world web pages
- Styling modern layouts with CSS
- Working with JavaScript DOM and events
- Building interactive frontend components without frameworks
